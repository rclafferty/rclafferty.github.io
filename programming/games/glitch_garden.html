Glitch Garden (Unity C#)

Glitch Garden was a 2D tower defense game developed as part of a tutorial on Udemy, which goes over basic 2D sprite imports, character movement, character animation, point system management, and more.

GitHub Link:
https://github.com/rclafferty/Glitch-Garden/

Roles:
UI design, event scripting, animation event scripting, collision

Designed for:
Windows Computer

Controls:
Mouse (left click to select, right click to rotate)

Requires Admin Privileges:
No

Actively Developing:
No

What have I learned from this?

I learned a lot about 2D animation and how to trigger events or methods from within an animation, such as firing a projectile at a given point in the animation. I also learned a lot about 2D design in general for that style of a project.

Is there anything I'd do differently?

I am still actively developing it when I have time and am continuing to learn.
