Laser Defender (Unity C#)

Laser Defender was a fun Unity C# game based on a Galaga-type influence that was developed as part of a tutorial on Udemy. It was used to learn 2D collisions, events, player controls, and more from a top-down shooter perspective.

GitHub Link:
https://github.com/rclafferty/Laser-Defender/

Roles:
UI implementations, Level Design, Character Select functionality, C# scripting, User Input, Material Design

Designed for:
Windows Computer

Controls:
Mouse left click to choose options, Arrow Keys or WASD to move, optional controller input

Requires Admin Privileges:
No

Actively Developing:
No

What have I learned from this?

I learned UI design, events, and how to dynamically change sprites to implement a rotation effect. I also learned how to dynamically spawn random sprites and animations that can be attached for cool gameplay effects.
