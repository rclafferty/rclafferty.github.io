Block Breaker (Unity C#)

GitHub Link:
https://github.com/rclafferty/Block-Breaker/

Roles:
UI implementations, Level design, C# scripting, user input, collision detection

Designed for:
Windows Computer

Controls:
Mouse left click to choose options, Arrow Keys or WASD to move, optional controller input

Requires Admin Privilages:
No

Actively Developing:
No

 

Block Breaker was developed as part of a Unity C# tutorial on Udemy. It teaches 2D collisions, events, triggers, physics, and more at a slower pace to help ease into 2D game development.

What have I learned from this?

I learned mostly about collision events in 2D, such as changing a sprite on collision.

Is there anything I'd do differently?

There are some unfinished portions that I could polish up from a UI/UX perspective.
