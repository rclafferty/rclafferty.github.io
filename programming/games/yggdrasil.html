Yggdrasil (Unity C#)

GitHub Link:
https://github.com/rclafferty/Yggdrasil/

Creating scripts to control user events, environment events (such as timing out), storyboarding, recording voice-over audio, assisting in level design

Designed for:
Windows Computer

Requires Admin Privileges:
No

Actively Developing:
No

 

Yggdrasil was a product game made for Kansas State University's 4th Annual Game Jam, created by myself and three others. On this team, there were two programmers, two artists, and all four of us contributed to designing the story, levels, and overall design. This project was created using the Unity game engine, Blender (for 3D models), and open-licensed audio and textures.

What have I learned from this?

I learned about designing terrains and collision problems involved with doing so. Terrains were a very large part of designing our main map. I also learned about particle effects to design multiple aspects of the game, including the fire effects on the title screen and the effects in the torch on the main map.

Is there anything I'd do differently?

We had a lot of GitHub issues that my team is better prepared to handle now. We also had a lot of model import issues and my team has since determined the root cause of those issues.
