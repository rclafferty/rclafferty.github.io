About

Wooded

I am a third year master's student at the University of Nebraska-Lincoln studying Computer Science. I also have a Bachelor's of Science in Computer Science from Kansas State University, where I focused my studies on cyber security and video game development. I am currently the president of the UNL Game Development Club and studying educational game development/gamification as my master's thesis under Dr. Stephen Cooper.

My interests include software development, game development, cyber security, artificial intelligence, web development, and more.

Wooded was created at the Kansas State University Annual Game Jam in February 2019 based on the theme "Left Behind." I worked alongside two other team members to develop this game using Unity 2018.3, with my primary roles including level design, UI design, event scripting, tilemap, and some art using Photoshop. This game features original art and original music, as everything in the game was created from scratch.

What have I learned from this?

I learned how to use Photoshop for developing the title screen text art ("Wooded", "Start", and "Quit"). I also learned about tilemaps and how to use them to implement the maze, the forest, and a river scene (not implemented in the game jam build).

Is there anything I'd do differently?

I would've researched more efficient use of tilemaps, but it wasn't bad for my first time.
